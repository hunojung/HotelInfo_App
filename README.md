# HotelInfo_App

2021-11-25 ~ 2021-12-03 playdata 16기 1차 팀 프로젝트 프론트엔드 소스코드 입니다.

야놀자, 여기어때와 같은 호텔 정보 조회 서비스를 구현하였습니다.

Back-end github 주소

https://github.com/hunojung/HotelInfo

## 사용한 프레임 워크

- Angular

## 프로젝트 실행

npm i g

npm i g --only=dev

npm i g @angular/cli

npm upadte

vsc 끄고 다시 킴

- 실행
ng serve --host 127.0.0.1 --port 4200
